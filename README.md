Simple-Calculator-Demo
=========================

Simple calculator demo using `react`, `redux`, `es2017`.  

## Installation

```
    Clone this repository
    Navigate to the directory in a terminal
    npm install
    npm start
    start developing
```

## How Does It Work?

For running a development build run:
`npm start`

For sample production build run:
`npm run deploy`
